创作动机

我有写博客的习惯，一开始的时候就像本教程前言所述的那样，非常麻烦的进行写作。在这个过程中我寻找能够简化该流程的工具。正值当时 Craft 推出了 Extension 功能，于是我写了 Craft 的插件进行发布，并为此迭代了好几个版本，包括但不限于：

  1. Github Action 构建 + OSS 上传图片，后来发现 Github Action 中的服务调用国内 OSS 接口总超时，于是换成本地。
  2. 本地构建 + OSS 上传图片，后来发现本地构建需要用到 Apple Script，于是写了一个 Xhelper.app 来接受来自 Craft Extension 的命令。
  3. 后来，Craft 官方不再维护 Extension，而且由于 Xhelper.app 这个 Apple Script 的存在，导致博客发布流程无法「众乐乐」，也无法在非 Mac 平台运行，于是这才转战到 Noition，调研一圈后发现，Notion database 正合我意！

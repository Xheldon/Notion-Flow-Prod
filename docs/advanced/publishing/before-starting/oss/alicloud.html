阿里云 OSS

阿里云对象存储 OSS（Object Storage Service）是一款海量、安全、低成本、高可靠的云存储服务，提供最高可达 99.995 % 的服务可用性。多种存储类型供选择，全面优化存储成本。官网地址见 阿里云 OSS。

提示

如果你没有阿里云 OSS 服务，你需要先 购买，创建之后使用 OSS 服务还需要开通该服务（多此一举了算是，我资源包都买了，难道还不会给我自动开通吗🤣）。我买的配置是 OSS 资源包、标准-本地冗余存储、中国内地通用、40GB、6 个月，花费 4.98 元（因为我没买加速流量包，单存储的价格其实跟腾讯云价格差不多）。

获取（创建） Bucket 和 Region

进入 阿里云对象存储 OSS 控制台，点击创建 Bucket，按照你的需求配置：

记下这个 Bucket 名称和区域，后面会用到。

提示

与腾讯云不同的是，阿里云的区域需要自己找对应区域的代码，并未直接在控制台给出，这里是 对照表。如上图中我的 Bucket 就是 notion-blog，区域是 华北2（北京），对应的代码是 oss-cn-beijing。

获取 AccessKey Id 和 AssessKey Secret

进入 阿里云 AccessKey 管理页面，点击继续使用 AccessKey（你也可以使用子账号，这里不做介绍），记录下 AccessKey Id 和 AccessKey Secret，后面会用到，请务必安全保存。

Github 配置

获取 Github Personal Token

进入 Github Personal 权限申请页面，填写备注信息和修改过期时间为永不过期，选择仓库权限范围为: repo

提示

为什么需要使用 Tokens(classic) 而不是 Fine-grained tokens?

原因有以下几个，如果你不介意的话，可以选择 Fine-grained tokens：

  • Fine-grained tokens 最多只能选择一年的有效期，这意味着每过一年你就需要重新设置一次 Notion Flow。
  • Fine-grained tokens 处于 Beta 阶段，可能会有不稳定的情况。

诚然，Fine-grained tokens 对于权限的控制更加细致，可以让你只授权 Notion Flow 访问你的某个仓库，从而更加安全。如果你选择使用 Fine-grained tokens，那么你需要授权你的博客仓库，然后选择 Contents 读写权限。 这些都取决于你自己。

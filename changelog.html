更新日志

Notion Flow 的版本变更记录。完整发布与下载见 GitHub Releases。

0.4.5

2026-06-21

新功能

  • 全新界面设计：侧边栏改为「目录 / 发布」分段，发布页「所见即所发」——文章卡片、必填字段（title/name/date）实时校验、连接状态、分步进度与精简日志。
  • 设置页重做为左侧分区导航（概览 / Notion / GitHub / 图片转存 / 目录&界面 / 高级 / 关于）+ 四步「设置向导」。
  • 图片转存支持「主存储 + 多镜像」：一张图并行上传到多个对象存储，主地址写入文章，镜像失败不影响发布。
  • 模块转换插件改为双栏编辑器：左侧写函数，右侧看「输入 block → 输出字符串」，可一键「运行测试」，覆盖 13 种可自定义区块类型。
  • 设置页「测试连接」一键校验 Notion / GitHub。
  • 点击图标行为优化：Notion 页直接打开侧边栏，非 Notion 页弹出引导。
  • 新增 notion.com 域名支持。

修复

  • 多处稳定性与边界问题修复。

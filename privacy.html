隐私政策

Notion Flow（下称「本扩展」）是一款由 Xheldon 开发、免费提供的浏览器扩展，用于把你在 Notion 中写好的内容发布到你自己的 GitHub 仓库。本扩展的核心设计原则是「你的数据始终由你掌控」——它是纯前端、非侵入的工具，作者不运营任何用于接收你数据的服务器。

一句话概括：

  • 作者不收集、不存储、不传输你的任何数据；本扩展不含分析、埋点、崩溃上报或任何遥测代码。
  • 你的 Token、密钥与设置只保存在你浏览器的扩展存储中，绝不发送给我们。
  • 本扩展仅在你主动操作时，用你提供的凭证从你的浏览器直接与你所配置的服务（Notion、GitHub、对象存储）通信。

我们收集什么

不收集任何信息。本扩展不包含任何分析、统计、崩溃上报或遥测代码，不会收集你的 IP 地址、设备信息、使用日志或任何个人身份信息，也没有任何用于接收这些信息的服务器。文档站 notion-flow.xheldon.com 上只有帮助文档与说明页面，是你点击链接后在浏览器中打开的普通网页；本扩展不会向该域名发送任何数据。

数据存储

为了正常工作，本扩展会在本地保存以下内容：你的 Notion 集成 Token、GitHub Token、对象存储（腾讯云 COS / 阿里云 OSS / AWS S3 / Cloudflare R2 等）的访问密钥，以及你的发布与界面设置。

这些数据通过浏览器的扩展存储接口（chrome.storage.sync）保存。它们保留在你的浏览器中；如果你开启了浏览器账号同步（如 Chrome / Edge 的同步功能），浏览器厂商会在你登录的设备之间同步这些数据。无论哪种情况，它们都不会被发送给作者。你可以随时在扩展的设置页清除这些数据，或卸载扩展以删除全部本地数据。

数据如何流动（第三方服务）

当你使用发布功能时，本扩展会用你自己提供的凭证，从你的浏览器直接与下列服务通信，全程不经过作者的任何服务器：

  • Notion API（api.notion.com）：读取你选择发布的页面内容与属性。
  • GitHub API（api.github.com）：将生成的 Markdown 提交到你指定的仓库。
  • 你所配置的对象存储 / CDN（腾讯云 COS、阿里云 OSS、AWS S3、Cloudflare R2 或兼容 S3 接口的自建服务）：转存页面中的图片与视频。

这些都是由你选择、并用你自己的账号与密钥连接的第三方服务。相关数据从你的浏览器直接发送给它们，它们各自的隐私政策适用于相应数据，作者无法访问也不承担其处理责任。

可选的 AIGC 功能

AIGC 是一项可选、默认关闭且仍在开发中的功能。如果你选择启用，需要自行提供 OpenAI（或兼容服务）的 API Key；相关文本会用你的 Key 直接发送给该 AI 服务，受其隐私政策约束。作者不参与、也不会接收其中的任何数据。

所需权限及用途

  • 网站访问权限：用于在 Notion 页面中读取目录结构，并直接访问 Notion、GitHub 以及你所配置的对象存储等接口。
  • 通知：用于显示发布进度与结果的本地通知。
  • 存储：用于保存上述设置与凭证。

本扩展不会在与上述功能无关的网站上读取或修改你的数据。

儿童隐私

本扩展不面向 13 岁以下儿童，也不会有意收集任何儿童的个人信息（实际上，本扩展不收集任何人的个人信息）。
